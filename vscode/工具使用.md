##### 1: 快捷键
```
1：CTRL + 方向键的右键 ：移到单词尾部
2：CTRL + SHIFT+右键：选择单词
3：ALT  + 鼠标左键拖动: 选择一列或者多列
4: SHIFT + 方向键： 选择光标到行尾

Ctrl+Shift+N         打开一个新窗口 
Ctrl+N               新建文件 
Alt+Up 或 Alt+Down   上下移动一行 
Ctrl+End             移动到文件结尾 
Ctrl+Home            移动到文件开头 
Ctrl+Delete          删除光标右侧的所有字 
Ctrl+D               删除当前行 
Ctrl+H               查找替换 
```
